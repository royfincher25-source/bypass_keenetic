## ФАЗА 1: Критические исправления

### Задача 1.1: Рефакторинг unblock_ipset.sh

**Цель:** Устранить дублирование, добавить batch load, DNS кэш

**Метрики:**
- До: 10,000 IP → ~30 сек
- После: 10,000 IP → ~1-2 сек

**Подзадачи:**
1. Создать функцию `process_list(ipset_name, input_file)` - убрать дублирование кода
2. Реализовать `ipset restore -!` для пакетной загрузки вместо O(n) вызовов ipset add
3. Добавить DNS кэш в `/tmp/dns_cache` с TTL 3600 сек
4. Добавить таймаут DNS запросов (3 сек)
5. Устранить shell injection - использовать переменные вместо system()

**Файл:** deploy/router/unblock_ipset.sh

**Ожидаемый результат:**
- Сокращение строк кода с 162 до ~80
- Ускорение загрузки в 10-15 раз
