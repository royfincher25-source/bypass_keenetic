## 📊 ФУНКЦИИ

### 1. **Команда `/ipset` — Статистика ipset** ⭐⭐⭐

**Приоритет:** Высокий  
**Время:** 5-10 минут

**Описание:**
- Показывает количество IP в каждом ipset
- Быстрая проверка актуальности списков

**Пример вывода:**
```
📊 Статистика ipset:

unblocksh: 143 IP
unblocktor: 7 IP
unblockvless: 26 IP
unblocktroj: 7 IP
```

**Реализация:**
- Файл: `src/bot3/handlers.py`
- Функция: `ipset_stats(message)`
- Команда: `@bot.message_handler(commands=['ipset'])`

---

### 2. **Команда `/traffic` — Трафик сервисов** ⭐⭐⭐

**Приоритет:** Высокий  
**Время:** 10-15 минут

**Описание:**
- Показывает трафик через каждый сервис
- Парсинг iptables counters

**Пример вывода:**
```
📊 Трафик:

🟢 Shadowsocks: 80.45 MB
🔷 VLESS: 5668.12 MB
🟠 Tor: 0.38 MB
🟣 Trojan: 0.00 MB
```

**Реализация:**
- Файл: `src/bot3/handlers.py`
- Функция: `traffic_stats(message)`
- Команда: `@bot.message_handler(commands=['traffic'])`

---

### 3. **Кнопка "🔄 Обновить ipset"** ⭐⭐⭐

**Приоритет:** Высокий  
**Время:** 15-20 минут

**Описание:**
- Кнопка в сервисном меню
- Запускает `/opt/bin/unblock_ipset.sh`
- Показывает результат обновления

**Пример:**
```
⏳ Обновление ipset...

✅ ipset обновлён!

unblocksh: 150 IP (+7)
unblocktor: 8 IP (+1)
unblockvless: 30 IP (+4)
unblocktroj: 7 IP (0)
```

**Реализация:**
- Файл: `src/bot3/menu.py` — добавить кнопку
- Файл: `src/bot3/handlers.py` — обработчик callback
- Скрипт: `/opt/bin/unblock_ipset.sh`

---

### 6. **Авто-уведомления** ⭐

**Приоритет:** Низкий  
**Время:** 30 минут

**Описание:**

**Пример уведомления:**


**Реализация:**
- Файл: `src/bot3/main.py`
- Функция: `monitor_services()`
- Запуск: threading.Thread (фон)
