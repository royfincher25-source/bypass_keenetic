## 2. Компоненты VLESS

### 2.1 Парсер ключей

**Файлы:**
- ✅ `bot3/utils.py` (строки 354-393)
- ✅ `core/parsers.py` (строки 14-75)

**Функция:** `parse_vless_key()`

**Возможности:**
- ✅ Парсинг VLESS ключей (vless://...)
- ✅ Поддержка всех параметров (security, encryption, pbk, fp, sni, sid, spx, flow)
- ✅ Валидация параметров (security: none/tls/reality)
- ✅ Валидация портов (1-65535)
- ✅ Кэширование результатов (1 час)
- ✅ Обработка ошибок через декоратор `@notify_on_error()`

**Пример ключа:**
```
vless://uuid@server.com:443?security=reality&encryption=none&pbk=...&fp=chrome&sni=...&sid=...&spx=/&flow=xtls-rprx-vision
```

---

### 2.2 Генерация конфигурации

**Файлы:**
- ✅ `bot3/utils.py` (строки 515-525)
- ✅ `vless_template.json`

**Функция:** `vless_config()`

**Параметры:**
- `localportvless` — локальный порт (по умолчанию 10810)
- `address` — адрес сервера
- `port` — порт сервера
- `id` — UUID пользователя
- `security` — тип безопасности
- `encryption` — шифрование
- `pbk` — публичный ключ
- `fp` — fingerprint
- `sni` — Server Name Indication
- `sid` — Short ID
- `spx` — SpiderX
- `flow` — поток

**Шаблон:** `vless_template.json`
- ✅ Корректная структура Xray
- ✅ Поддержка VLESS Reality
- ✅ Правильные плейсхолдеры

---

### 2.3 Конфигурационный файл

**Путь:** `/opt/etc/xray/config.json`

**Генерация:**
```python
vless_config(key, bot, chat_id)
```

**Структура:**
```json
{
  "log": { ... },
  "inbounds": [
    {
      "port": {{localportvless}},
      "protocol": "dokodemo-door",
      "sniffing": { ... }
    }
  ],
  "outbounds": [
    {
      "tag": "vless-reality",
      "protocol": "vless",
      "settings": { ... },
      "streamSettings": { ... }
    }
  ]
}
```

---

### 2.4 Сервис Xray

**Скрипт:** `/opt/etc/init.d/S24xray`

**Порт:** 10810 (настраивается в `.env`)

**Команды:**
```bash
/opt/etc/init.d/S24xray start
/opt/etc/init.d/S24xray stop
/opt/etc/init.d/S24xray restart
/opt/etc/init.d/S24xray status
```

---

### 2.5 Интеграция с обходом

**ipset таблица:**
```bash
ipset create unblockvless hash:net -exist
```

**dnsmasq правила:**
```bash
# Генерируется unblock_dnsmasq.sh
ipset=/$line/unblockvless
server=/$line/127.0.0.1#40500
```

**Список доменов:**
```bash
/opt/etc/unblock/vless.txt
```
