## 🚀 Как использовать

### Быстрый старт:

```bash
# 1. Создать .env
cp .env.example .env

# 2. Заполнить токен и логин
nano .env  # или любой редактор

# 3. Проверить
python check_config.py

# 4. Установить зависимости
pip install -r requirements.txt

# 5. Запустить тесты
pytest tests/ -v

# 6. Проверить линтинг
make lint
```
